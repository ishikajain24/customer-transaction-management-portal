# Customer Transaction Management Portal

A full-stack business management web application designed to streamline customer and transaction records for jewelry businesses. The system enables secure customer management, daily transaction tracking, and automatic balance calculations for gold, silver, and cash transactions.

## 🚀 Features

### 🔐 Secure Authentication
- JWT-based authentication
- Password hashing using bcrypt
- Authorized access for business owner/admin only

### 👥 Customer Management
- Add, update, and delete customer records
- Maintain a centralized customer database
- Quick customer search and retrieval

### 💰 Transaction Management
- Record daily transactions
- Track:
  - Gold transactions
  - Silver transactions
  - Cash transactions
- Maintain transaction history for each customer

### 📊 Automated Balance Calculations
- Pending gold balance tracking
- Pending silver balance tracking
- Cash advance management
- Metal deposit calculations
- Real-time balance updates

### 📈 Dashboard & Analytics
- Customer transaction overview
- Balance summaries
- Organized transaction records
- Responsive and user-friendly interface
